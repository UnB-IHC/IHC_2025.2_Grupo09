# Ferramentas de Acessibilidade

Para auxiliar e facilitar o desenvolvimento de soluções acessíveis, foram selecionadas algumas **ferramentas de acessibilidade** que podem apoiar as etapas de análise, teste e melhoria de projetos digitais.

- [Accessibility Test Framework for Android](https://github.com/google/Accessibility-Test-Framework-for-Android): framework de testes de acessibilidade para aplicações Android.  
- [WAVE – Web Accessibility Evaluation Tool](https://wave.webaim.org/): ferramenta para avaliação automática de acessibilidade em páginas web.  
- [DynoMapper – Visual Sitemap Generator](https://dynomapper.com/): gera sitemaps visuais e realiza auditorias de acessibilidade.  
- [JAWS – Leitor de Telas para Windows](https://www.tecassistiva.com.br/catalogo/jaws/): leitor de telas amplamente utilizado por pessoas com deficiência visual.  
- [ACHECKS – Caixa de Ferramentas para Acessibilidade](https://www.achecks.org/): conjunto de ferramentas para verificação de conformidade com diretrizes de acessibilidade.  
- [AXE – Accessibility Engine for Automated Web UI Testing](https://github.com/dequelabs/axe-core): mecanismo de testes automatizados de acessibilidade para interfaces web.  
- [Lighthouse – Google](https://developer.chrome.com/docs/lighthouse): ferramenta de auditoria integrada ao Chrome para avaliar desempenho, SEO e acessibilidade.  
- [WebAIM – Contrast Checker](https://webaim.org/resources/contrastchecker/): verificador de contraste entre cores de texto e plano de fundo.
